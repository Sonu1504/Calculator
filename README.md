# Calculator 

## Overview
A simple browser-based calculator built using **HTML, CSS, and JavaScript**. The application provides a clean calculator interface with support for basic arithmetic operations, parentheses, percentages, and decimal values.

## Features
- Addition, subtraction, multiplication, and division
- Parentheses support: `(` and `)`
- Percentage operator (`%`)
- Decimal number calculations
- Clear (`C`) button to reset input
- Interactive button-based input system
- Custom styled calculator UI
- Background image support

## Functionality

### User Interface
- Calculator display area for showing expressions and results.
- Numeric buttons (0–9).
- Arithmetic operators (+, -, ×, ÷).
- Parentheses support.
- Percentage operator.
- Clear button (`C`).
- Equals button (`=`) for evaluation.

### Working Logic
1. User clicks calculator buttons.
2. Values are appended to the display.
3. Clicking `=` evaluates the mathematical expression.
4. Clicking `C` clears the display.

## Usage

- Click the buttons to enter a mathematical expression.
- Press `=` to calculate the result.
- Press `C` to clear the display and start a new calculation.
